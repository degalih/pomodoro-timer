Kebijakan Privasi Pomodoro Timer

Terakhir Diperbarui: 1 Maret 2026 
+1

Selamat datang di Pomodoro Timer. Kami menghargai privasi Anda dan berkomitmen untuk melindungi data pribadi Anda. Kebijakan Privasi ini menjelaskan bagaimana informasi Anda dikelola saat menggunakan aplikasi mobile kami pada platform iOS dan Android.
+1

1. Pengumpulan Data
Aplikasi ini dirancang dengan prinsip privasi sebagai prioritas utama.


Data Pribadi: Kami tidak mengumpulkan informasi identitas pribadi seperti nama, alamat email, nomor telepon, atau lokasi.


Data Penggunaan: Kami tidak menggunakan alat analitik pihak ketiga untuk melacak perilaku Anda dalam aplikasi pada versi MVP ini.

2. Penyimpanan Data Lokal
Semua data yang dihasilkan melalui aplikasi disimpan secara lokal di perangkat Anda. Data ini meliputi:


Pengaturan Aplikasi: Kustomisasi durasi timer, preferensi suara, target harian, dan pengaturan mode gelap.
+4


Statistik Produktivitas: Riwayat jumlah sesi Pomodoro yang diselesaikan dan total waktu fokus selama 7 hari terakhir.


Catatan: Karena data disimpan secara lokal, menghapus aplikasi atau menghapus data aplikasi melalui pengaturan sistem akan mengakibatkan data statistik dan pengaturan Anda hilang secara permanen.

3. Izin Perangkat
Untuk menjalankan fungsinya dengan optimal, aplikasi memerlukan izin berikut:


Notifikasi: Diperlukan untuk memberi tahu Anda saat sesi fokus atau waktu istirahat telah berakhir, bahkan saat aplikasi berjalan di latar belakang.
+2


Audio & Getar: Diperlukan untuk memainkan alarm suara dan memberikan umpan balik haptik saat timer selesai.
+2


Latar Belakang (Background Execution): Diperlukan agar timer tetap berjalan akurat saat Anda menggunakan aplikasi lain atau saat layar terkunci.
+1

4. Keamanan Data
Karena semua data tetap berada di perangkat Anda dan tidak dikirim ke server luar, risiko kebocoran data pihak ketiga sangat minimal. Kami menyarankan Anda untuk menjaga keamanan perangkat Anda sendiri dengan menggunakan fitur pengamanan standar (seperti passcode atau biometrik).

5. Layanan Pihak Ketiga
Aplikasi ini tidak terintegrasi dengan layanan pihak ketiga, akun media sosial, atau penyedia iklan pada versi MVP ini.

6. Perubahan Kebijakan Privasi
Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu seiring dengan pengembangan fitur baru (seperti sinkronisasi cloud di masa depan). Perubahan akan diinformasikan melalui pembaruan aplikasi.

7. Hubungan Kami
Jika Anda memiliki pertanyaan mengenai kebijakan ini, Anda dapat menghubungi tim pengembang kami melalui saluran komunikasi internal yang tersedia.
